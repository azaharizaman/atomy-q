# Supported Flows

## Buyer Flows Supported In Alpha

- tenant company registration
- tenant user login
- RFQ creation and edit
- RFQ line-item definition
- vendor invite management
- quote upload and review
- normalization conflict resolution
- comparison readiness and finalization
- award creation and signoff
- decision-trail verification
- minimal tenant-scoped user invite and status management

## Flows Explicitly Not Supported

- vendor self-service portal journeys
- partner-facing billing and subscription management
- advanced report authoring
- negotiation workflow execution
- risk workflow execution
- non-alpha dashboard/navigation modules that are currently hidden or deferred
